# Cloud File Sharing Platform

A cloud-based file sharing application built using FastAPI, Docker, AWS S3, and AWS EC2.

## Features

- File upload and download
- AWS S3 storage integration
- Dockerized deployment
- Hosted on AWS EC2

## Tech Stack

- Python
- FastAPI
- Docker
- AWS S3
- AWS EC2
